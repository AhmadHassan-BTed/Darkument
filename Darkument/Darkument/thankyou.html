Enable File Access

To use Darkument with local PDF files on your computer,
you need to enable file URL access for the extension.

1

Open Extensions Page

Click Here, or Navigate to chrome://extensions (Chrome) or edge://extensions (Edge) in your browser.

2

Find Darkument

Look for the Darkument extension in your list of installed extensions.

3

Enable File Access

Click on "Details" and toggle on "Allow access to file URLs". This permission is required for Darkument to work with local PDF files.

4

You're Ready!

Once enabled, you can open any local PDF file and click the Darkument extension icon to toggle dark mode.

🎉 What's Next?

After enabling file access, try opening a PDF file from your computer. Click the Darkument icon in your browser toolbar to instantly switch to dark mode!
