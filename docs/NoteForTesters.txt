This Chrome extension adds a toggleable visual overlay that inverts webpage colors—primarily for use with PDFs or light-background content—by applying a full-screen, white div with mix-blend-mode: difference. When you click the extension icon, it either adds or removes this overlay on the current tab, simulating a dark mode without altering the underlying content. The extension tracks the overlay state per tab and updates the icon to reflect whether the overlay is active (icon1.png) or inactive (icon2.png). Smooth transitions and console logs confirm actions for testers. The overlay is non-interactive (pointer-events: none) and cleans up automatically when tabs are closed. Just an extension that inverts the theme.
